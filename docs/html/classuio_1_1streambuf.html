Detailed Description

A simple byte-buffer.

This class implements an efficient and lightweight byte-buffer.

Attention
setbuf must be called before this class can be used.

void* uio::streambuf::dump ( )
inline

Get the entire contents of the buffer.

Returns a pointer to the entire contents of the buffer. The contents of the buffer will be overwritten on the next call to sputc or sputn (this that calls to size() are valid until the next call to sputc or sputn).

Get the number of bytes in the buffer.

This function is similar to in_avail except it returns the total number of bytes that have been put into the buffer (as opposed to in_avail which returns the number of bytes put into the buffer minus internal cursors position).
