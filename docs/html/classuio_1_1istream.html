Detailed Description

An input data object.

istream is an object that supports the following operations:

  • gcount
  • get
  • sync
Note
Input data might not be automatically synchronized. Users should call sync before get.

virtual istream& uio::istream::operator>> ( char *  s)
inlinevirtual

Convenience function for writing input data to string.

gcount characters are copied to s, as well as a null character. This means that gcount plus 1 bytes are copied to s.

Attention
This function has no safe-guards to check that s is large enough to store the entire input data streams contents.

[out]sFirst address of character array.

virtual istream& uio::istream::sync ( )
pure virtual

Pure virtual function to synchronize (update) input data stream.

This function updates the internal buffers (i.e. does whatever needs to happend to bring the internal buffers up to date).

Note
It might be necessary to call this function before get.
